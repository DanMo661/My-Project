# STM32 嵌入式开发技能 (Claude Code Skill)

通用的 STM32 嵌入式固件开发技能，支持 Keil MDK + STM32CubeMX HAL 库开发流程。

## 特性

- **自动检测**: 从 .ioc 文件自动识别芯片型号和外设配置
- **多系列支持**: 支持 STM32 F0/F1/F4/F7/G0/G4/H7/L0/L4/WB 等全系列
- **完整开发流程**: 从项目创建、代码编写到重构优化
- **外设支持**: I2C、SPI、UART、FreeRTOS、PWM、ADC 等
- **代码风格**: 遵循中文注释、early return、模块化设计规范

## 适用场景

- 新建 STM32 项目
- 重构已有代码（代码太乱、太臃肿）
- 设计串口通信协议
- 外设驱动开发和调试

## 技术栈

- **IDE**: Keil MDK (μVision)
- **代码生成**: STM32CubeMX
- **HAL 库**: STM32 HAL Driver
- **调试工具**: ST-Link、J-Link
- **操作系统**: 支持 FreeRTOS 集成

## 使用方式

在 Claude Code 中触发此技能，支持以下场景：

```
# 新建项目
> 创建一个 STM32F407 的 I2C 传感器读取项目

# 重构代码
> 帮我重构这个外设库，代码太乱了

# 设计协议
> 设计一个串口通信协议，用于上位机和下位机通信

# 调试问题
> I2C 通信总是超时，帮我排查一下
```

## 开发规范

技能内置以下开发规范：

1. **项目检测**: 必须先找到 .ioc 文件，确认硬件配置
2. **硬件确认**: 写代码前确认引脚、总线参数、中断方式
3. **代码风格**: 中文注释、无 `__` include guard、early return 模式
4. **模块化**: 外设驱动独立封装，参考 `ads1115库/` 风格

## 支持的外设

| 外设 | 协议 | 用途 |
|------|------|------|
| I2C | 2线 | 传感器、EEPROM、显示模块 |
| SPI | 4线 | 高速通信、Flash、显示屏 |
| UART | 2线 | 串口调试、上位机通信 |
| ADC | 模拟 | 电压采集、传感器读数 |
| PWM | 数字 | 电机控制、LED 调光 |
| FreeRTOS | - | 实时任务调度 |
